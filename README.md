# Kitsap Mobile Brakes & Oil

Static marketing website for Kitsap Mobile Brakes & Oil, prepared for GitHub Pages.

## Pages

- Home: `/`
- Services and pricing: `/services/`
- Contact: `/contact/`

## Local preview

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

The private/offline invoicing tool and business spreadsheet are intentionally not included in this public website repository.
